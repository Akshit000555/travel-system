# Smart Travel Recommendation System 2.0

Improved UI with better responsiveness and Render compatibility.

## Deploy on Render

**Build Command:**
```
pip install -r requirements.txt
```

**Start Command:**
```
python -m streamlit run app.py --server.port=$PORT --server.address=0.0.0.0
```

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
